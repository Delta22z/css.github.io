Guía de CSS

CSS (Cascading Style Sheets) es un lenguaje utilizado para describir la presentación de un documento HTML. CSS puede controlar el diseño y el formato de múltiples páginas web a la vez.

Propiedades Adicionales de CSS

Además de las propiedades básicas, CSS ofrece muchas otras propiedades que puedes usar para mejorar tu página web. Aquí hay algunas adicionales:

  • text-align: Define la alineación del texto.
  • font-weight: Define el grosor de la fuente.
  • border-radius: Define los bordes redondeados de un elemento.
  • text-transform: Controla la capitalización del texto.
  • box-shadow: Agrega sombra a los elementos.

Ejemplo de Mejora del Diseño con CSS

A continuación se muestra cómo CSS puede mejorar el diseño de una página web:

Propiedad Descripción Ejemplo
text-align Define la alineación del texto. text-align: center;
font-weight Define el grosor de la fuente. font-weight: bold;
border-radius Define los bordes redondeados de un elemento. border-radius: 10px;
text-transform Controla la capitalización del texto. text-transform: uppercase;
box-shadow Agrega sombra a los elementos. box-shadow: 0 4px 8px rgba(0, 0, 0, 0.2);

Comentarios en CSS

Los comentarios en CSS se usan para dejar notas y no se muestran en la página web. Se escriben de la siguiente manera:

/* Este es un comentario en CSS */

Conclusión

CSS es una herramienta poderosa para mejorar el diseño y la presentación de tus páginas web. Con un conocimiento básico de las propiedades y cómo usarlas, puedes crear sitios web visualmente atractivos y profesionales.
